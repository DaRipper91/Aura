# Aura // Desktop Edition (Go)
Aura is a high-performance, local AI agent optimized for Linux, Windows, and macOS.
This repository contains the Desktop CLI (Aura-Go), built with Go and Charm BubbleTea.

### Ecosystem
- **Aura-Desktop (This Repo):** Go-native TUI.
- **Aura-Xbox:** Rust-native engine for UWP.
- **Aura-APK:** Kotlin + Rust (JNI) for Android.
